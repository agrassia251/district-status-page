I am using Internet Identity for admin access. Here are some instructions to gain access as an admin

The admin login uses Internet Identity — here's how to set it up:

Go to your app's /admin route in the browser
You'll see a login prompt with a "Login" button
Click it and you'll be redirected to the Internet Identity flow
You can sign in using a passkey, Google, Apple, or Microsoft account
Once authenticated, you'll have full access to the admin panel to create and update incidents
For your other techs:

Share the /admin URL with them
Each tech logs in with their own Internet Identity account
All authenticated users get admin access — no separate role system needed for an internal tool
First time using Internet Identity?

Visit identity.ic0.app to create an anchor/account first, then use that to log in to your admin panel.
